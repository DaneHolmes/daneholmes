Projects

Affixionary

Affixionary is a website that serves as an etymological dictionary of English affixes. Currently, it only contains prefixes, but I hope to soon add suffixes and bases. I also hope to rebuild it soon and possibly implement a free, open API.

Crypto-Dash

Crypto-Dash is a shiny app framework mostly written in tidyverse R. It automatically preforms basic financial analysis of live cryptocurrency data and displays the information through a variety of graphs. Crypto-Dash is still being improved in how it display trends in the most common cryptocurrencies.
